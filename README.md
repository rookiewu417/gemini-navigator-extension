# Gemini Navigator (Gemini 提问导航扩展)

Gemini Navigator 是一款专为 Google Gemini Web 版设计的浏览器扩展程序。它会在页面的右下方添加一个精美的、采用毛玻璃（Glassmorphism）设计风格的悬浮导航栏，自动提取您在当前会话中所有的提问，帮助您在长篇的对话中快速定位并跳转到之前的上下文。

## ✨ 主要功能

- **🔍 自动提取提问**: 智能识别并提取您发送给 Gemini 的所有问题，生成清晰的提问列表。
- **⚡ 快速跳转**: 点击列表中的任意问题，页面将平滑滚动至该提问所在位置，并带有短暂的视觉高亮提示，方便您快速回顾上下文。
- **💎 极致的视觉体验**: 采用现代化的磨砂玻璃（毛玻璃）UI 设计方案，支持浅色与暗色模式的自适应，提供精致、丝滑的交互与悬停动画。
- **📐 智能布局防遮挡**: 展开导航栏时，扩展会自动动态调整 Gemini 聊天主内容区域的右边距，为导航栏留出专用空间，**彻底告别导航栏遮挡 AI 长篇回答**的烦恼。
- **🪶 轻量无感**: 无需复杂配置，即装即用。收起状态下仅展示一个精巧的圆形悬浮按钮（FAB），绝不干扰您的正常阅读视野。

## 🛠️ 安装说明 (Edge/Chrome)

由于本插件尚未上架浏览器应用商店，请通过“开发者模式”进行本地解包安装：

1. 下载本项目的文件夹到您的电脑本地。
2. 打开您的 Microsoft Edge 或 Google Chrome 浏览器。
3. 在地址栏输入 `edge://extensions/` (Edge) 或 `chrome://extensions/` (Chrome) 并访问。
4. 在页面左下角或右上角找到并开启 **“开发人员模式” (Developer mode)**。
5. 点击 **“加载解压的扩展” (Load unpacked)**。
6. 指定并选择本插件所在的文件夹（即包含 `manifest.json` 的那一级目录）。
7. 安装完成！刷新或重新打开 [Gemini 官网](https://gemini.google.com/) 即可体验全新的导航功能。

## 💡 使用方法

1. 打开 Gemini 网页版，并在任意会话中进行正常的文字交流。
2. 页面右下角会自动出现一个圆形的菜单按钮，点击即可展开侧边**提问导航**面板。
3. 面板中会按对话顺序列出您当前会话发出的所有问题（长问题会自动提取摘要摘要）。
4. 点击任意列表项，页面即刻平滑滚动到对应的提问气泡位置。
5. 在侧边栏展开时，Gemini 的文章内容会自动向左收缩避让；点击右上角的 "X" 关闭侧边栏后，文章排版又会恢复原状。

## 📁 项目结构

- `manifest.json` - 浏览器扩展的清单配置文件。
- `content.js` - 核心逻辑脚本。负责解析页面的 DOM 节点提取提问、使用 MutationObserver 监听对话更新，并动态控制页面主元素的右侧外边距（Margin-Right）防遮挡。
- `styles.css` - UI 样式表。包含所有的毛玻璃特效、滚动条美化、动画过渡效果以及亮/暗模式自适应代码。
